# PopX Account App

A mobile-first React implementation of the supplied PopX assessment screens, created as an original submission with English interface copy.

## Live Demo

[https://reactjsintern.vercel.app](https://reactjsintern.vercel.app)

## Screens

- Welcome
- Sign in
- Create account
- Account settings

## Run Locally

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Tech Stack

- React
- React Router
- Vite
- CSS
